# Скрипт генерации отчётов

# Для запуска скрипта требуется создать экземпляр класса "ReportGenerator"

# И передать ему следующие параметры:

- header: заголовок документа
- cur_rk_path: путь к файлу "Текущая РК"
- org_path: путь к файлу "Органический трафик"
- groups_path: путь к файлу с данными о группах
- campaigns_path: путь к файлу с данными о кампаниях
- prev_rk_path: путь к файлу с данными предыдущей РК
- outlier_rate: множитель, отвечающий за величину отклонения данных, которые будут считаться выбросом по умолчанию = 1.5
  чем выше данный множитель, тем большее значение различия между данными будет считать выбросом, и наоборот
